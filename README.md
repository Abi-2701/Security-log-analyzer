# Security Log Analyzer

## Overview

Security Log Analyzer is a Python-based cybersecurity tool that analyzes log files and identifies potential security incidents such as failed login attempts, warnings, and critical system errors.

The project demonstrates fundamental concepts of log monitoring, security event detection, and risk assessment.

## Features

- Analyze security log files
- Detect failed login attempts
- Count warning events
- Count critical errors
- Generate basic risk assessment
- Command-line interface

## Technologies Used

- Python 3
- File Handling
- Conditional Logic
